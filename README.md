# 🛍️ UrbanStyle - Fashion E-Commerce Platform

Welcome to **UrbanStyle**, a modern and responsive fashion e-commerce web application designed to provide users with a seamless online shopping experience. The platform enables customers to browse fashion products, explore categories, and manage their shopping cart efficiently.



## 📌 Project Title

**UrbanStyle – Fashion E-Commerce Website**

---

## 💻 Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* JavaScript
* PHP
* MySQL
* Git & GitHub
* Visual Studio Code

---

## ✨ Features

### 🏠 Home Page

* Responsive navigation bar
* Featured products section
* Modern and user-friendly interface

### 🛍️ Product Management

* Display products in responsive cards
* Product images, names, and prices
* Categories and subcategories
* Product filtering by category

### 🛒 Shopping Cart

* Add products to cart
* View cart items
* Update product quantities
* Remove products from cart
* Clear cart functionality
* Total price calculation

### 📱 Responsive Design

* Mobile-friendly interface
* Responsive product grid layout
* Optimized for different screen sizes
